Graft

Your local environment, everywhere.

Keep your editor, your terminal, your dotfiles. Graft connects any remote machine or container to your local environment. Run commands, sync files, forward ports.

main.go
package main

import (
    "net/http"
    "runtime"
)

func main() {
    http.HandleFunc("/", handler)
    http.ListenAndServe(":8080", nil)
}

func handler(w http.ResponseWriter, r *http.Request) {
    if runtime.GOOS == "darwin" {
        w.Write([]byte("hello from darwin"))
    } else {
        w.Write([]byte("hello from linux"))
    }
}

Why not just SSH?

Your editor stays local

Edit in VS Code, Vim, Emacs, whatever. Files sync to the remote automatically. No remote editor, no scp, no rsync cron.

Commands are transparent

Forward go, make, or cargo. They run on the remote but look local. Same working directory, same output.

Ports just work

Remote servers show up on localhost. No ssh -L tunnels, no port flags to remember.
